Command line interface (CLI)Â¶

To launch the Linguistica CLI:

$ linguistica cli

The command line interface works for either a corpus text file or a wordlist file. Parameter changes are supported. Results are saved as text files.

A sample CLI session, with a corpus text file as input and default settings for all parameters and options:

$ linguistica cli
